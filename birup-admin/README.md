# briup-admin

21级山西农大，2024.9.9，开始编写项目。分为`briup-admin`、`briup-ui`、`briup-serve`。

1. `briup-admin`: 后台管理系统。
2. `briup-ui`: 门户网页
3. `briup-serve`: 后台服务

### 安装依赖
```
npm install
```

### 运行项目
```
npm run serve
```

### 打包项目
```
npm run build
```

### 项目介绍

`briup-admin`是后台管理系统，采用`vue2` + `vue-router` + `vuex`以及`element-ui`组件库和`g2plot`图表库，以及其他插件等完成。
